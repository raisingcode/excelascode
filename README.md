## Backstory
Excel As Code was concieved as ask some on-prem sysadmin moving some vm to public cloud.  The main requirement  was to be able to create a list of vms and deploy them for in azure in without having the patients or deep understanding of devops and cloud migration best practices. After several rounds or pitches on different was to start cloud native and with devops Excel as Code began as simple worksheet to help teams rapid add new vms and get them deployed .
